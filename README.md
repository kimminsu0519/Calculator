## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/pyqt6-calculator.git
    cd pyqt6-calculator
    ```

2. Install required packages:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the app:
    ```bash
    python main.py
    ```
